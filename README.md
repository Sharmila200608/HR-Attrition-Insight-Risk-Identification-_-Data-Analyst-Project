# 🏢 HR Attrition Insight & Risk Identification

## 📌 Project Overview

This project focuses on analyzing employee attrition using data-driven techniques to help organizations understand why employees leave and how to improve retention strategies.

The analysis is performed using **Excel, Python, and Power BI**, covering data cleaning, exploratory data analysis (EDA), and interactive dashboard creation.

---

## 🎯 Problem Statement

Organizations often face high employee attrition, leading to increased hiring costs and reduced productivity.
This project aims to identify key factors influencing attrition and provide actionable insights for HR decision-making.

---

## ⚙️ Tools & Technologies Used

* 📊 **Excel** – Data cleaning, preprocessing, and pivot analysis
* 🐍 **Python (Pandas, NumPy, Matplotlib, Seaborn)** – Data analysis & visualization
* 📈 **Power BI** – Interactive dashboards and business insights

---

## 📂 Dataset Information

* Dataset: IBM HR Analytics Employee Attrition Dataset
* Total Employees: 1470
* Features include:

  * Age, Gender, Department
  * Monthly Income, Job Role
  * Work-Life Balance, Job Satisfaction
  * Overtime, Years at Company
  * Attrition (Target Variable)

---

## 🔄 Project Workflow

### 1️⃣ Data Cleaning (Excel)

* Removed unnecessary columns
* Renamed columns for clarity
* Created new features:

  * Salary Band (Low / Medium / High)
  * Attrition Flag (0 / 1)
* Generated pivot tables for initial insights

---

### 2️⃣ Exploratory Data Analysis (Python)

* Handled missing values and duplicates
* Performed statistical analysis
* Created visualizations:

  * Distribution plots
  * Box plots
  * Heatmaps
  * Pair plots
* Identified correlations between features

---

### 3️⃣ Dashboard Development (Power BI)

Built an interactive dashboard with:

* 📊 Attrition KPIs
* 📉 Department-wise analysis
* 📈 Salary and job role insights
* 🔍 Key Influencers visual (AI-based insights)
* 🧠 Smart Narrative (auto insights)
* 💬 Q&A visual for natural language queries

---

## 📊 Key Insights

* 🔥 Employees working overtime are more likely to leave
* 💰 Low salary employees show higher attrition
* 🏢 Sales department has the highest attrition rate
* 👥 Age group 25–35 shows maximum employee turnover
* 😊 Job satisfaction and work-life balance strongly impact retention

---

## 💡 Recommendations

* Reduce employee overtime workload
* Improve salary structure for low-income employees
* Enhance work-life balance initiatives
* Focus on employee engagement programs
* Provide career growth opportunities
